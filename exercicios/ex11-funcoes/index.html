Conversão de Pixels para rem

Crie uma função Sass chamada px-para-rem que converte valores de pixels para rems com base em um tamanho base. Use a função para definir o tamanho de fonte de um elemento com 24 pixels em rems, assumindo um tamanho base de 16 pixels.
